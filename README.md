# TOTP-PilotProject
TOTP 파일럿 프로젝트


## 개발 환경
IDE : IntellJ
Framework : Spring

## 개발 기간
2020-09-07 ~ 2020-09-11

## 개요 
1. ID , PW로 1차 로그인 진행
2. 로그인 되면 ID와 Gauth 를 통해 QR코드를 만들어 웹 화면 출력함
3. 출력된 QR코드를 GOTP 앱을 통해 스캔한후 OTP 코드를 발급받음
4. 발급된 OTP 코드를 입력하면 2차 로그인 완료

## 추후 개발
